# Luxury Clothing Brand Website

A modern, premium, and fully responsive luxury fashion eCommerce website built with **React, Vite, TypeScript, and Tailwind CSS**.

Inspired by international fashion brands, this project delivers a clean, elegant, and high-performance shopping experience with a luxury user interface.

## ✨ Features

- Premium Luxury UI/UX Design
- Fully Responsive Layout
- Modern Navigation
- Home Page
- Shop Page
- Men's Collection
- Women's Collection
- Outdoor Collection
- Accessories
- Product Details
- Shopping Cart
- Secure Checkout
- Wishlist
- Compare Products
- Search Overlay
- Customer Account
- Blog
- About Us
- Contact Page
- Smooth Animations
- Fast Performance
- SEO Friendly Structure
- Built with React + Vite + TypeScript + Tailwind CSS

## 🛠 Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Context API
